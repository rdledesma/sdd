¿Por qué este curso?

La IA puede escribir código en segundos. Pero si los requerimientos son ambiguos, el resultado sigue siendo incorrecto.

Spec-Driven Development propone invertir el proceso tradicional:

Spec → Proposal → Tasks → Código
